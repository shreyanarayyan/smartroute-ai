# Welcome to SmartRoute AI project
SmartRoute AI is an AI-powered delivery route optimization web application built for an AI Essentials project.

The system helps users find the most efficient delivery route by organizing multiple delivery locations into an optimized order.

## Features

- Add multiple delivery locations
- Optimize delivery route
- Estimate travel distance
- Predict delivery time
- Calculate fuel cost
- Delivery priority handling
- Route efficiency score
- Modern dashboard interface

## Tech Stack

### Frontend
- React
- Vite
- TypeScript
- Tailwind CSS
- Lovable AI generated UI

### Backend
- Node.js
- Express.js
- REST API

## Project Goal

The goal of SmartRoute AI is to reduce delivery time, fuel usage, and travel distance using simple AI-based route optimization logic.
